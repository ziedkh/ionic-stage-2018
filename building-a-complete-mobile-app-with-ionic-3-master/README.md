## Installation

Install  dependencies
```sh
npm install
```

## Run the app on the browser

```sh
ionic serve
```

## Run the app on your phone

```sh
ionic cordova platform add android
ionic cordova run android
```

or

```sh
ionic cordova platform add ios
ionic cordova run ios
```
